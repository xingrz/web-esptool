[Web ESPTool](https://esp.xingrz.me)
==========

[![license][license-img]][license-url] [![issues][issues-img]][issues-url] [![commits][commits-img]][commits-url]

Flash Espressif chips on your web browser!

## Development

```sh
npm install
npm run serve
```

## Features / TODOs

- [x] Auto program
- [x] Stub loader
- [x] ESP8266/ESP8285
- [x] ESP32
- [ ] ESP32-S2
